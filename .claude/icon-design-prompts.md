# 梧桐工具箱 - AI图标设计方案

## 📋 设计要求

### 基本规格
- **尺寸**: 1024x1024像素
- **格式**: PNG (透明背景)
- **分辨率**: 300 DPI
- **风格**: 扁平化设计、简洁现代

### 平台要求
1. 在16x16像素下依然清晰可辨
2. 在不同背景下都有良好对比度
3. 支持暗色和亮色主题

## 🎨 方案一：梧桐叶工具箱

### 设计描述
A modern minimalist tool box icon with chinese parasol tree leaves, clean flat design, primary color deep forest green with golden yellow accents, white background, 1024x1024px, vector style, no shadows, high contrast, recognizable at small sizes

### 中文提示词

- 🎨 配色：深绿色 + 金黄色 + 白色

生成1024x1024后，需要生成以下尺寸：

### 必备尺寸
```

256x256.png  - 高DPI显示
512x512.png  - 商店展示

icon.icns    - macOS ICNS文件
```
