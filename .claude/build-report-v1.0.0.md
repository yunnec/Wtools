## 🎯 安装说明

### 用户安装步骤

#### MSI安装包
1. 双击 `wutong-toolbox_0.1.0_x64_en-US.msi`
2. 按照安装向导提示操作
3. 选择安装路径（默认：`C:\Program Files\wutong-toolbox\）
4. 完成安装
5. 从开始菜单或桌面快捷方式启动

#### NSIS安装包
1. 双击 `wutong-toolbox_0.1.0_x64-setup.exe`
2. 选择安装组件（建议全选）
3. 选择安装路径
4. 点击"安装"
5. 等待安装完成
6. 可选择"立即运行"启动应用

### 静默安装
```bash
# MSI静默安装
msiexec /i wutong-toolbox_0.1.0_x64_en-US.msi /quiet /norestart

# NSIS静默安装
wutong-toolbox_0.1.0_x64-setup.exe /S
```
