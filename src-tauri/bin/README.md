# 236日志解压工具依赖文件

本目录需要以下文件才能正常工作：

## 必需文件
- **DecryptLogForWinX64.exe** - 车机日志解密工具

## 说明
1. 请将 `DecryptLogForWinX64.exe` 放在本目录中
2. 该文件应来自原始的236解压工具包
3. 如果没有此文件，解密功能将无法工作

## 注意事项
- 该文件是Windows可执行文件，仅支持Windows平台
- Linux/macOS用户可能需要使用Wine或其他兼容层运行
- 请确保文件来源可靠，以避免安全风险

## 获取方式
请联系原始工具的开发者或系统管理员获取此文件。
