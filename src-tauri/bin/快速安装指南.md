# 🔧 236日志解密工具 - 快速安装指南

## 🚨 问题现象
如果您看到以下错误：
```
❌ 未找到解密工具 DecryptLogForWinX64.exe
```

## ✅ 解决方法（3步搞定）

### 第1步：获取解密工具
- 联系您的 **系统管理员** 或 **IT 部门**
- 说明需要 `DecryptLogForWinX64.exe` 文件
- 这是 236日志解压功能专用的解密工具

### 第2步：放置文件
将 `DecryptLogForWinX64.exe` 复制到以下任一位置：

**推荐位置**：
```
应用安装目录/bin/DecryptLogForWinX64.exe
```

**备选位置**：
```
应用安装目录/DecryptLogForWinX64.exe
```

### 第3步：重启应用
- 关闭梧桐工具箱
- 重新启动应用
- 236日志解压功能现在应该可以正常工作了

## ❓ 常见问题

**Q: 我从哪儿获取 DecryptLogForWinX64.exe？**
A: 联系您的系统管理员或IT部门，这是内部工具，不对外分发。

**Q: 可以从网上下载吗？**
A: 不建议。请务必使用官方提供的解密工具，确保安全性。

**Q: 我放在其他位置可以吗？**
A: 可以。应用会在两个位置查找：
   - `bin/DecryptLogForWinX64.exe` （推荐）
   - `DecryptLogForWinX64.exe` （备选）

**Q: 重启后还是不行怎么办？**
A: 请检查：
   1. 文件名是否正确：`DecryptLogForWinX64.exe`
   2. 文件是否放在正确位置
   3. 联系 IT 部门确认文件版本
